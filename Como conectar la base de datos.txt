Cómo conectar la base de datos.

1- Instalar y abrir el archivo en visual:
2- Una vez abierto el archivo, escribir en la terminal de VS code 
	cd backend
	npm install express mysql2 dotenv
3- Abrí XAMPP → iniciá Apache y MySQL
Entrá a http://localhost/phpmyadmin
Click en "Nueva" → escribí maply_services → Crear
Si tenés el importar el archivo SQL 
4- confirmar que el .env diga lo siguiente.
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=
DB_NAME=maply_services
DB_PORT=3306

5- escribir en la terminal para levantar el servidor:
cd src
node app.js

y deberias ver 

Servidor corriendo en el puerto 3000
Conexión exitosa a la base de datos MySQL (maply_services)
